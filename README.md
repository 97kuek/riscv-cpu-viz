# RISC-V CPU 可視化ツール

RISC-V プロセッサの5段パイプラインにおける信号フローをインタラクティブに可視化する Web アプリケーションです。

## スクリーンショット

（後で追加予定）

## 概要

このツールは、RISC-V アーキテクチャの基本的な命令がCPU内をどのように処理されるかを視覚的に学習できるように設計されています。各パイプラインステージで活性化する配線・モジュール・制御信号をリアルタイムで確認できます。

## 使い方

1. 画面上部の **命令タブ** から学習したい命令を選択します
2. **ステージタブ** またはキーボードの **← →** キーでパイプラインステージを切り替えます
3. SVGダイアグラム上で青くハイライトされたモジュール・配線がそのステージで活性化している部分です
4. 右パネルでシグナル値・説明文・ALU内部図を確認できます

## 対応命令一覧

| 命令 | 種別 | 動作 |
|------|------|------|
| `add x1, x2, x3` | R型 | x1 ← x2 + x3 = 4 + 3 = 7 |
| `lw x1, 8(x2)` | I型 | x1 ← mem[x2+8] = mem[12] = 0xDEADBEEF |
| `sw x3, 8(x2)` | S型 | mem[x2+8] ← x3 = 3 |
| `beq x1, x2, L` | B型 | x1==x2? → 1≠4 なので分岐なし |

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS v3
- **図形描画**: Pure SVG（アニメーションライブラリ・react-flow 不使用）

## ローカル開発手順
